#### Custom Port

To run on a different port:

**Docker Compose**:
```yaml
services:
  masonic-time:
    ports:
      - "8080:5005"  # Change 8080 to your desired port
    environment:
      - PORT=5005    # Keep internal port as 5005
```

**Manual Installation**:
```bash
PORT=8080 npm start
```

### Reverse Proxy Setup

#### Nginx Example

```nginx
server {
    listen 80;
    server_name masonic.yourdomain.com;

    location / {
        proxy_pass http://localhost:5005;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

#### Apache Example

```apache
<VirtualHost *:80>
    ServerName masonic.yourdomain.com
    
    ProxyPreserveHost On
    ProxyPass / http://localhost:5005/
    ProxyPassReverse / http://localhost:5005/
</VirtualHost>
```
